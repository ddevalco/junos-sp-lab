## To create this Lab
You have the following or higher versions need to be  installed:
* VirtualBox 4.3.36 
* Ansible 2.3.0
* Vagrant 1.9.1

## To work with this lab
* Clone this repo
* Run "vagrant up"
* Run "vagrant status" to see list of network devices

## To reset this lab 
* Run "vagrant destroy"
* Run "vagrant up" to start using it again
